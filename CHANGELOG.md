# 更新日志

## v1.0.0 (2026-04-02)

### 新增功能

#### 后端 (Cloud Functions)
- **MySQL 数据库层**
  - 管理员用户表 (admin_users)
  - 租户表 (tenants) - 支持授权状态、联系人信息
  - 设备指标表 (device_metrics) - 完整支付宝 IoT 数据字段
  - 同步日志表 (sync_logs) - 记录每日同步任务
  - 导出记录表 (export_logs)

- **支付宝 SDK**
  - 授权 URL 生成
  - Token 换取和刷新
  - 设备数据批量查询
  - 全量分页遍历

- **云函数**
  - `gateway` - API 网关（Koa 路由 + JWT 认证）
  - `authCallback` - 支付宝授权回调
  - `dataSync` - 定时数据同步（每日凌晨 2 点）
  - `initAdmin` - 管理员初始化

#### 前端 (React)
- **管理员后台**
  - 租户管理（创建、编辑、启用/禁用、删除）
  - 授权链接生成和复制
  - 全局数据概览（统计卡片 + 趋势图）
  - 同步状态监控
  - 手动同步触发
  - 管理员密码修改

- **租户端**
  - 数据大屏（核心指标 + 交易趋势 + 省份分布）
  - 设备数据列表（多条件筛选 + 分页）
  - 图表分析（趋势图、饼图、柱状图）
  - 报表导出（Excel/PDF/Word）
    - 完整数据导出
    - 汇总分析报告
    - 设备排名报告
    - 门店分析报告
  - 同步状态查看
  - 密码修改

### 部署配置
- `cloudbaserc.json` - CloudBase 部署配置
- `deploy.sh` - 一键部署脚本
- `.env.example` - 环境变量模板
- `README.md` - 完整部署文档
