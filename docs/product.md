# 产品文档

## 产品概述

**名称**: 支付宝 IoT 数据分析 SaaS 系统 (alinfc-iot-saas)

**定位**: 支付宝 IoT 设备数据多租户 SaaS 分析平台，帮助商户管理和分析 IoT 设备数据。

**目标用户**: 使用支付宝 IoT 设备的商户和管理员

## 核心功能

### 已完成功能 ✅

| 功能模块 | 说明 | 状态 |
|---------|------|------|
| 用户认证 | 管理员/租户登录、JWT 认证 | ✅ 已上线 |
| 租户管理 | 创建、编辑、启用/禁用租户 | ✅ 已上线 |
| 数据概览 | 统计卡片、趋势图 | ✅ 已上线 |
| 数据导出 | Excel/PDF/Word 导出 | ✅ 已上线 |

```
前端: React 18 + TypeScript + Vite + Ant Design
后端: Vercel Serverless Functions + Supabase
部署: Vercel
监控: Sentry + 企微通知
```
